# Портфолио

Личная страница-портфолио с перечнем проектов.

## Что умеет

- Показывает список проектов с описаниями и ссылками
- Плавная прокрутка по навигации
- Анимация появления секций
- Прогресс-бар прокрутки страницы
- Адаптивная вёрстка - нормально выглядит на телефоне

## Технологии

HTML, CSS, JavaScript (без библиотек), Font Awesome (иконки), Fetch API (загрузка JSON)

## Запуск

Открыть `index.html` через локальный сервер (например Live Server в VS Code) - нужен для загрузки JSON. Через двойной клик не сработает из-за политики CORS.
